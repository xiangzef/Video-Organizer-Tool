视频文件整理器 使用说明
================================

版本：v1.0
创建日期：2026年3月28日

一、程序文件说明
----------------
1. video_organizer.py - 命令行版本（基础功能）
2. 视频整理器.bat - 命令行版本启动文件
3. video_organizer_gui.py - GUI图形界面版本
4. 视频整理器_GUI.bat - GUI版本启动文件
5. 使用说明.txt - 本说明文件

二、系统要求
------------
1. Windows 7/8/10/11 操作系统
2. Python 3.6 或更高版本
3. 对于GUI版本，需要安装Tkinter支持

三、安装Python（如未安装）
------------------------
1. 访问 https://www.python.org/downloads/
2. 下载最新版Python安装程序
3. 安装时务必勾选 "Add Python to PATH"
4. 对于GUI版本，安装时选择安装Tkinter支持

四、使用方法
------------

A. 命令行版本（推荐给高级用户）
------------------------------
1. 双击运行 "视频整理器.bat"
2. 按提示操作
3. 输入或拖拽目录路径
4. 确认后开始整理

B. GUI图形界面版本（推荐给普通用户）
-----------------------------------
1. 双击运行 "视频整理器_GUI.bat"
2. 点击"浏览..."按钮选择目录
3. 点击"开始整理"按钮
4. 查看处理进度和日志

五、功能特点
------------
1. 支持多种视频格式：
   - 常见格式：.mp4, .avi, .mkv, .mov, .wmv, .flv
   - 其他格式：.webm, .m4v, .mpg, .mpeg, .3gp, .ts等

2. 智能处理：
   - 递归查找所有子文件夹中的视频文件
   - 自动将视频文件移动到一级目录
   - 智能重命名避免文件名冲突
   - 安全删除空文件夹（仅删除完全空的文件夹）

3. 安全保护：
   - 不会删除原始目录
   - 不会删除非空文件夹
   - 操作前有确认提示
   - 详细日志记录

六、注意事项
------------
1. 请先备份重要数据
2. 程序不会删除非视频文件
3. 如果目标文件已存在，会自动添加数字后缀
4. 只能删除完全空的文件夹（无任何文件或子文件夹）
5. 处理大量文件时可能需要一些时间

七、常见问题
------------
Q: 程序提示"未找到Python"怎么办？
A: 请先安装Python并确保已添加到系统PATH。

Q: GUI版本无法启动怎么办？
A: 请确保Python安装了Tkinter支持，或使用命令行版本。

Q: 程序会删除我的文件吗？
A: 程序只会移动视频文件和删除完全空的文件夹，不会删除其他文件。

Q: 如何恢复操作？
A: 程序没有撤销功能，请在使用前备份重要数据。
